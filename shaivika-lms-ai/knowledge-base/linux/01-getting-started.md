# Linux System Administration: Getting Started

Welcome to the Linux System Administration course!

## What is Linux?
Linux is a family of open-source Unix-like operating systems based on the Linux kernel. It is the leading operating system on servers and mainframe computers, and is the foundation for the Android OS.

## Why Learn Linux?
As an AI Engineer or Full Stack Developer, a vast majority of the servers, cloud instances, and containers you will interact with run Linux. Understanding how to navigate, configure, and troubleshoot Linux environments is a critical skill.

## Prerequisites
- Access to a Linux environment (Virtual Machine, WSL, or a remote server)
- Curiosity to learn the command line!
