# Git & GitHub: Getting Started

Welcome to the Git and GitHub foundation course!

## What is Version Control?
Version control systems (VCS) are software tools that help software teams manage changes to source code over time. Git is the most widely used modern version control system in the world today.

## What is GitHub?
GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere.

## Prerequisites
- A terminal/command line interface
- Basic understanding of files and directories
